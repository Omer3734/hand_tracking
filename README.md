Poğaça yerken yağlanan ellerimle bilgisayar fareme dokunmak istemediğim zamanlarda kullandığım, MediaPipe ve OpenCV paketleriyle çalışan cursor controller yazılımı.
-
Fare kontrolü için sağ elinizi kullanmanız ve elinizin kamera tarafından algılanması yeterli. İşaret parmağının ucu, cursor konumu olarak ayarlanacaktır.
